 # OldUR — Old Unix Revive

OldUR (Old Unix Revive) is a custom Linux distribution built from scratch using Linux From Scratch (LFS).

Its main goal is to revive the classic UNIX and early Linux experience while maintaining modern stability and performance.

OldUR focuses on simplicity, retro aesthetics, and full system control.

---

## Features

### Classic UNIX Interface
- NsCDE (Common Desktop Environment style)
- Retro window management
- Terminal-first workflow
- Minimal desktop environment

### Built From Scratch
- Based on Linux From Scratch (LFS)
- Custom-built toolchain
- Manual system compilation
- Full control over binaries

### Custom Package System
- Own package manager (in development)
- Wrapper over base system tools (early stage)
- Simple dependency handling
- Lightweight repositories

### Retro Design
- Classic color schemes
- UNIX-style fonts
- Minimal icons
- Old-school wallpapers

### System Philosophy
- No bloatware
- No unnecessary services
- Performance-focused
- Transparency-first design

---

## Project Goals

OldUR aims to:

- Preserve classic UNIX aesthetics
- Teach low-level Linux system building
- Provide a clean and minimal OS
- Encourage learning and experimentation
- Build a community around retro computing

This project is both educational and experimental.

---

## Why OldUR?

Modern Linux distributions are often heavy and complex.

OldUR focuses on:

✔️ Control  
✔️ Simplicity  
✔️ Speed  
✔️ Learning  
✔️ Retro culture  

It is designed for users who want to understand how Linux really works.
